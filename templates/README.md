# ESP-IDF Templates

This directory contains template files for ESP-IDF projects generated by the agent.

## Contents

- `esp_idf/`: ESP-IDF specific template files
  - `esp32s3_box_lcd_config.h`: LCD configuration header for ESP32-S3-BOX
- `dht11/`: DHT11 temperature and humidity sensor driver
  - `dht11.h`: DHT11 driver header file with function declarations and definitions
  - `dht11.c`: DHT11 driver implementation for reading temperature and humidity data
## Usage

These template files are automatically included in generated projects when appropriate based on the design requirements and platform specifications.
